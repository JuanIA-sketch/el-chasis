# {{nombre}}

Proyecto Web/Vite generado con El Chasis — estructura, hooks de seguridad y TDD listos desde el primer commit.

## Estructura

- `src/` — código fuente de la app
- `public/` — assets estáticos que Vite sirve tal cual
- `tests/` — tests (TDD rojo→verde con Vitest)
- `vite.config.ts` — config mínima de Vite, lista para sumarle plugins
- `.claude/` — config de Claude Code con El Freno de Mano y El Paracaídas ya instalados

## Cómo arrancar

Instalá Vite y tu framework preferido (`npm create vite@latest` como referencia de dependencias), o sumá `vite` y `vitest` a tu `package.json` y usá esta estructura tal cual.
