# El Chasis

Generador de scaffolding para nuevos proyectos con buenas prácticas ya instaladas: estructura de carpetas, hooks de seguridad, README con licencia y tests listos para TDD.

## Estado
En desarrollo — parte del reto Juegos Imperiales (Imperio Agéntico).

## Instalación
```
npm install
npm test
```

## Uso (planeado)
```
el-chasis create
```

Ver `BRIEF-EL-CHASIS.md` para el contexto completo del proyecto.
